## 🛠️ 使用方法

请搭配 [93@Home](https://github.com/SaltWood-Studio/Open93AtHome-V3) 使用

1. 克隆此仓库到本地

```bash
git clone https://github.com/Mxmilu666/93Home-Dash.git
```

2. 进入项目目录并安装依赖：
```bash
cd 93Home-Dash
npm install
```

3. 编译并热重载以进行开发
```bash
npm run dev
```

4. 编译并压缩以进行生产部署
```bash
npm run build
```
